# VSCODE PLAYGROUND

## When you want to code simple, do it in vscode!

- `C++`, `Python`, `Typescript`, `Javascript`

### requirements

> Node, npm, python, g++ must be installed.

### getting start

- `yarn ts` : debug ts script
- `yarn js` : debug js script
- `yarn python` : debug python script
- `f5` : debug cpp file
