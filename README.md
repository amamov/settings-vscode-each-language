# VSCODE PLAYGROUND

## When you want to code simple, do it in vscode!

- `Typescript`, `Python`, `C++`, `Java`, `Javascript`

### requirements

> Node, npm, python, g++, jdk must be installed.

### getting start

- `yarn ts` : debug ts script
- `yarn js` : debug js script
- `yarn python` : debug python script
- `f5` : debug cpp file
